Bootstrap 排版

Bootstrap 使用 Helvetica Neue、 Helvetica、 Arial 和 sans-serif 作为其默认的字体栈。

使用 Bootstrap 的排版特性，您可以创建标题、段落、列表及其他内联元素。

引导主体副本

为了给段落添加强调文本，则可以添加 class="lead"，这将得到更大更粗、行高更高的文本，如下面实例所示

.lead {
    margin-bottom: 20px;
    font-size: 16px;
    font-weight: 300;
    line-height: 1.4;
}
@media (min-width: 768px) {
    .lead {
        font-size: 21px;
    }
}

强调

标签/类 描述 原理
<small> 设置文本为父文本大小的 85% 
.small, small {
    font-size: 85%;
}
<strong> 设置文本为更粗的文本 
b, strong {
    font-weight: 700;
}
<em> 设置文本为斜体 
/* user agent stylesheet */
i, cite, em, var, address, dfn {
    font-style: italic;
}
.text-left 向左对齐文本 
.text-left {
    text-align: left;
}
.text-center 居中对齐文本 
.text-center {
    text-align: center;
}
.text-right 向右对齐文本 
.text-right {
    text-align: right;
}
.text-muted 本行内容是减弱的 
.text-muted {
    color: #777;
}
.text-primary 本行内容是 primary 
.text-primary {
    color: #337ab7;
}
.text-success 本行内容是 success 
.text-success {
    color: #3c763d;
}
.text-info 本行内容是 info

.text-warning 本行内容是 warning

    color: #8a6d3b;
}
.text-danger 本行内容是 danger 
.text-danger {
    color: #a94442;
}

缩写

HTML 元素提供了用于缩写的标记，比如 WWW 或 HTTP。 Bootstrap 定义 <abbr> 元素的样式为显示在文本底部的一条虚线边框，当鼠标悬停在上面时会显示完整的文本（只要您为 <abbr> title 属性添加了文本）。 为了得到一个更小字体的文本，请添加 .initialism 到 <abbr>。
