1. 相关

百度开放云文档服务DOC（Document Service）
  1. 特点
    • 文档处理技术: 基于百度文库强大的文档转码、渲染技术，将源文档转换成HTML5，在线浏览时最大程度保留源文档样式
    • 文档处理功能: 文档元信息提取、缩略图生成、转码成功回调、离线浏览
    • 多端支持: 支持PC/WAP浏览器、Android、iOS主流平台，文档内容根据不同终端，自适应流式重排。
    • 文件存储在百度服务器
  2. 格式
    • 源文档支持doc、docx、ppt、pptx、xls、xlsx、vsd、pot、pps、rtf、wps、et、dps、pdf、txt、epub多种格式

2. 开始

  1. 准备
    • 下载: Java SDK
    • Access Key (AK): 点击查看
    • jdk1.6及更高版本
