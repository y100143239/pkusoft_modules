2. 安装maven

参考：官网
  1. 要求
    • JDK1.7 及以上（最好使用 jdk-7u79-macosx-x64.dmg ）
  2. 下载
    • 地址
    • 文件：apache-maven-3.3.9-bin.zip
  3. 环境变量 
    // 1. 创建 ~/.bash_profile 文件

// 2. 编辑

    M3_HOME=/usr/local/maven/apache-maven-3.3.9

    PATH=$M3_HOME/bin:$PATH

    # 不设置 JAVA_HOME，会报错 “Unsupported major.minor version 51.0”
    export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.7.0_79.jdk/Contents/Home

    export M3_HOME
    export PATH

// 3. 编译

    $ source ~/.bash_profile
  4. 修改本地仓库位置
    • 文件：/usr/local/maven/apache-maven-3.3.9/conf/settings.xml
    • 第55行添加 <localRepository>/Users/forwardNow/develop/mvnrep</localRepository>

3. 安装Eclipse插件

参考：Maven 实战系列之Eclipse插件与web项目
  1. 在线安装（行不通）
    • m2e
    • http://m2eclipse.sonatype.org/sites/m2e
  2. 离线安装
    • 下载： 地址
    • 解压后目录结构 
      maven3
    |-- eclipse
        |-- features
        |-- plugins
    • 配置到Eclipse 
      1. 在eclipse目录下的dropins目录下新建一个maven.link文件
2. 编辑 maven.link：（指向插件maven3）
    path=../soft/maven3
3. 重启Eclipse
    • 设置maven安装目录(Eclipse: Preferences -> Maven -> Installations)
    • 设置settings.xml(Eclipse: Preferences -> Maven -> User Settings)

4. 创建maven项目

参考：Maven 实战系列之Eclipse插件与web项目
  1. New Maven Project - Select project name and location
    • 勾选：Use default Workspace location
  2. New Maven Project - Select an Archetype
    • 选：maven-archetype-webapp
  3. New Maven Project - Enter a group id for the artifact
    • Group Id: net.pkusoft
    • Artifact Id: framework
