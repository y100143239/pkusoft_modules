    |-- .modal-dialog
        |-- .modal-content

    position: fixed;
    top: 0;

    bottom: 0;

5. 用法

5.1 激活（初始化）

1）通过元素激活

<!--
    data-toggle="modal"     ： 指定为 modal触发器
    data-target="#modal-1"  ： 指定触发的目标对象（通过CSS选择器）
-->
<button type="button" data-toggle="modal" data-target="#modal-1">激活</button>

<div class="modal fade" id="modal-1">
    <div class="modal-dialog">
        ...
    </div>
</div>

2）通过JavaScript

$('#modal-1').modal( [options] );

<div class="modal fade" id="modal-1">
    <div class="modal-dialog">
        ...
    </div>
</div>
参数名 参数值类型 默认值 描述
keyboard boolean true 键盘上的 esc 键被按下时关闭模态框。
backdrop boolean / string true true：点击遮罩层时会关闭modal，遮罩层有样式（黑色半透明）
false：点击遮罩层时会关闭modal，遮罩层无样式（完全透明）
"static"：点击遮罩层时不会关闭modal，遮罩层有样式（黑色半透明）
show boolean true 模态框初始化之后就立即显示出来。

5.2 方法

方法名 描述
$( "#modal-1" ).modal( [ options ] ) 激活（初始化）模态框，参数请参考上面的表格 
jQuery( "#modal-1" ).modal( {
    backdrop: "static"
} );
$( "#modal-1" ).modal( 'toggle' ) 手动切换模态框的状态（打开或关闭）。
在触发 shown.bs.modal 或 hidden.bs.modal 事件之前
$( "#modal-1" ).modal( 'show' ) 手动打开模态框。
在触发 shown.bs.modal 事件之前
$( "#modal-1" ).modal( 'hide' ) 手动关闭模态框。
在触发 hidden.bs.modal 事件之前

5.3 事件

事件类型 描述 
$('#myModal').on('show.bs.modal', function (e) {
    // do something...
})
show 方法调用之后立即触发该事件。如果是通过点击某个作为触发器的元素，则此元素可以通过事件的 relatedTarget 属性进行访问。 
$('#myModal').on('shown.bs.modal', function (e) {
    // do something...
})
此事件在模态框已经显示出来（并且同时在 CSS 过渡效果完成）之后被触发。如果是通过点击某个作为触发器的元素，则此元素可以通过事件的 relatedTarget 属性进行访问。 
$('#myModal').on('hide.bs.modal', function (e) {
    // do something...
})
hide 方法调用之后立即触发该事件。 
$('#myModal').on('hidden.bs.modal', function (e) {
    // do something...
})
此事件在模态框被隐藏（并且同时在 CSS 过渡效果完成）之后被触发。
