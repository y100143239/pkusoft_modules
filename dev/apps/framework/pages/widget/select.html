1. 基本示例

简单

1. 单选（默认值为第一项）
2. data：数据集
3. id：<option>的“value”属性值
4. text：<option>的文本内容（显示的值）
5. spell：搜索的关键字（text和spell都是关键字） 
<select class="form-control" id="select2_1"> 
<script>
jQuery("#select2_1").select2({
    data:[
        { id: 0, text: '张三', spell: "zs" },
        { id: 1, text: '李四', spell: "ls" },
        { id: 2, text: '王五', spell: "ww" },
        { id: 3, text: '赵六', spell: "zl" },
        { id: 4, text: '钱七', spell: "qq" }
    ]
});
</script>

占位符

1. 单选（默认值为占位符）
2. 添加<option value=""></option>，才会在初始化时显示占位符
3. placeholder：占位符的内容
4. allowClear：允许删除选中的条目，删除后显示占位符 
<select class="form-control" id="select2_2">
    <option><!-- 显示占位符的内容 --></option>
</select>
<script type="text/javascript">
jQuery("#select2_2").select2({
    data:[
        { id: 10, text: '张三', spell: "zs" },
        { id: 1, text: '李四', spell: "ls" },
        { id: 2, text: '王五', spell: "ww" },
        { id: 3, text: '赵六', spell: "zl" },
        { id: 4, text: '钱七', spell: "qq" }
    ],
    placeholder: "请选择",
    allowClear: true
});
</script>

“data-*” 的方式设置初始化参数

初始化时的参数设置，可通过 “data-*” 的方式 
<select class="form-control" id="select2_3"
        data-placeholder="select"
        data-allow-clear="true">
    <option value=""></option>
    <option value="1">壹</option>
</select>
<script> jQuery( "#select2_3" ).select2(); </script>

必选（自定义参数）

1. 必选（required）
2. required：扩展属性，指定没有选中时的CSS类
3. 下拉框关闭后执行校验 
<select class="form-control" id="select2_4"
        data-placeholder="select"
        data-allow-clear="true"
        data-required="validate-invalid">
    <option value=""></option>
    <option value="1">壹</option>
    <option value="2">贰</option>
</select>
<script>jQuery( "#select2_4" ).select2();</script>

2. 字典

字典（自定义参数）

xmlurl：扩展属性，指定XML字典文件的URL

搜索关键字
  • 编码（code），如 “140822”
  • 内容（text），如 “呼和浩特”
  • 拼音头（spell），如 “hbspqx”
<select class="form-control" id="select2_5"
        data-placeholder="请选择"
        data-allow-clear="true"
        data-required="validate-invalid"
        data-xmlurl="./pages/widget/DIC_CODE.xml">
    <option value=""></option>
</select>
<script>
    jQuery( "#select2_5" ).select2();
</script>
